ADG-Project

Этот проект создовался как площадка для развития навыков web-разработки, наработки различных практик, а также для их демонстрации. Идея заключалась в том, чтобы при написании проекта взаимодействовать с различным стеком web-технологий и внедрять разные "фишки", так сказать "Набить руку".

Больше конкретики

Поднятие проекта, его структуризация. Установка и настройка Webpack, Eslint, Stylelint, Babel, SCSS, Vue.js, JEST, работа с Node.js. Стек по проекту: HTML (Семантическая верстка, адаптивность, кросбраузерность, мобаил фест и доступность), CSS (Флекс-бокс, анимация), SCSS (Переменные, миксины, вложенность, импорты, и т.д.), SVG (Svg-анимацию, оптимизация и спрайты. Все svg-иллюстации созданы в программе Adobe Illustrator или написаны вручную.), Java Script (ES6 и различные "фишки" нативного JS), Vue.js, тестирование (JEST и Vue Test Utils).

И так, добро пожаловать!
